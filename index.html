Before you start — install Ollama

The app uses a local AI model (via Ollama) to rewrite résumé sections. Install it once, then launch the app — it pulls the model it needs automatically.

First launch

🍎 macOS
  • Unzip, then right-click ATS-Resume-Scorer.app → Open → Open.
  • (One time only — it's an unsigned app.)
  • It opens in your browser automatically.
🪟 Windows
  • Double-click ATS-Resume-Scorer-windows.exe.
  • If SmartScreen appears: More info → Run anyway.
  • Keep the small console window open while you use the app.
